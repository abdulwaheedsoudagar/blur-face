# blur-face
Detect and blur face using caffemodel

Install necessary dependency. 

pip install cv2 <br>
pip install numpy
<br>
<br>

Steps to use - 
1. git clone https://github.com/abdulwaheedsoudagar/blur-face.git 
2. python blur_face.py --image image.jpg --factor 1
<br><br>
factor starts from 1 <br><br>
1 => more blur<br>
9 => more blur<br>
